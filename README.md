# ImageConverter PRO

A modern, privacy-focused web application for converting images to JPG format. All conversion happens directly in your browser - no data is uploaded to any server.

## Features

- Convert multiple image formats to JPG: WebP, AVIF, JFIF, PNG, GIF, BMP, TIFF, SVG
- 100% client-side processing - your images never leave your device
- Modern, responsive design with high-tech aesthetics
- Fast conversion with visual feedback
- Download converted images instantly

## How to Use

1. Visit [ImageConverter PRO](https://xm14.github.io/image-converter-pro/)
2. Drag & drop an image or click to browse
3. Click "Convert to JPG"
4. Download your converted image

## Supported Formats

- **WebP** → JPG
- **AVIF** → JPG  
- **JFIF** → JPG
- **PNG** → JPG
- **GIF** → JPG
- **BMP** → JPG
- **TIFF** → JPG
- **SVG** → JPG

## Privacy Guarantee

This application runs entirely in your browser using JavaScript. We do not:
- Upload your images to any server
- Store or collect your image data
- Share your images with third parties
- Track your usage patterns

Your privacy and data security are our top priorities.

## Technical Details

Built with:
- HTML5 Canvas API for image processing
- Vanilla JavaScript (no frameworks)
- Modern CSS with Flexbox/Grid
- Responsive design for all devices
- No external dependencies for conversion logic
